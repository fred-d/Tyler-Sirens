Tyler Sirens

Get a live feed of police dispatches for the greater Tyler, Texas

Show me!

This project was inspired by Hack Tyler's Siren Project which is no longer in a working state after the Tyler Police Department revamped their internal dispatching systems.

This project is currently in development and no working functionality is - or ever will be - guaranteed. If you find any issues or would like to help contribute, check out the GitHub Repo or drop us a line at tylerwebdev.io.

Disclaimer: The Tyler-Sirens uses only information that is freely available to the public. This project is intended for educational and analytical use. If you have any concerns, please contact the Tyler Police Department concerning their policies on public records and dispatch information.
